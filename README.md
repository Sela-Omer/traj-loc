# TrajLoc: Trajectory-Attention Localization for Multi-Object Motion Control

Omer Sela, Inbar Huberman-Spiegelglas, Michael Rotman, Sagie Benaim, Avi Ben-Cohen

> **Multi-object motion control in image-to-video diffusion** via per-object attention
> localization with Gaussian trajectory heatmaps and dedicated text tokens for trajectory
> and appearance, scaling to scenes with up to 20 interacting objects.

**Project page:** https://sela-omer.github.io/traj-loc/
